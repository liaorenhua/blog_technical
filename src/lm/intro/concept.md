# 基本概念

## RAG 
RAG: Retrieval Augmented Generation 主要用于解决大模型幻觉问题。 

## 大模型  
大模型: 模型参数数量超过10亿的模型。  

大模型余生具有： 上下文学习能力

大模型幻觉: LLM大语言模型输出看似合理内容但与实际内容错误、虚构信息。 


- 产生原因1: 训练数据有噪音   
- 产生原因2: 没有训练到特殊知识点  
- 概率生成而非事实检索(本质根源)  # 大模型训练方式决定了他生产行为是预测下一个最可能得词  
- 用户模糊、含糊、模型“猜”错方向
