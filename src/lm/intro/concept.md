# 基本概念

## RAG 
RAG: Retrieval Augmented Generation 主要用于解决大模型幻觉问题。 

## 大模型  
大模型: 模型参数数量超过10亿的模型。  

大模型余生具有： 上下文学习能力

大模型幻觉: LLM大语言模型输出看似合理内容但与实际内容错误、虚构信息。 


- 产生原因1: 训练数据有噪音   
- 产生原因2: 没有训练到特殊知识点  
- 概率生成而非事实检索(本质根源)  # 大模型训练方式决定了他生产行为是预测下一个最可能得词  
- 用户模糊、含糊、模型“猜”错方向  


## agent(智能体)  
大模型 Agent 是以大语言模型为核心大脑，融合思维推理、工具调度、状态记忆、环境交互能力的智能体系统，核心是让大模型拥有 “思考如何做事 + 实际动手做事” 
的能力，通过感知 - 规划 - 执行 - 反馈的闭环逻辑，完成传统大模型无法独立实现的复杂、个性化、实操性任务，是大模型落地产业级应用的关键形态。
